# `reset` UseCases

## basic usage

```git reset [--mixed | --soft | --hard | --merge | --keep] [-q] [<commit>]```

```git reset [-q] [<tree-ish>] [--] <pathspec>...```

```git reset [-q] [--pathspec-from-file [--pathspec-file-nul]] [<tree-ish>]```

```git reset --patch [<tree-ish>] [--] [<pathspec>...]```

## reset current staged files

```sh
git reset HEAD
```
